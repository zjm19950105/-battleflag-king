# UI 占位素材清单

最后更新：2026-05-10

本文档记录测试沙盒 UI 中暂时没有真实素材、后续需要补正式图标或头像的项目。

当前策略：先使用文字、色块或 emoji-like 文本作为占位，保证 UI 流程和布局可以继续验证；等美术资源稳定后，再逐步替换为正式图标、头像或统一风格的小图标。

## 后续替换原则

- 优先替换高频出现的角色头像、兵种图标、装备槽图标和属性图标。
- 同一类别内保持一致的尺寸、描边、配色和可读性。
- 图标语义应服务于测试沙盒 UI 的快速识别，不在图标中承载复杂规则说明。
- 替换正式素材时同步检查 UI 布局，避免图标尺寸变化造成文字遮挡或控件抖动。
